# Практическая работа по ООП № 2
## Запуск
### В ручном режиме
```bash
python3 app.py
```

### Запуск c выводом в файл
```bash
python3 app.py -o "./result.txt"
```

### Запуск c пользовательским количеством измерений 
```bash
python3 app.py -d 4
```

### Запуск c указанным количеством фигур 
```bash
python3 app.py -c 1
```

## Данные типы фигур
### Треугольник
```python
0 or "Треугольник"
```

### Прямоугольник
```python
1 or "Прямоугольник" or "Квадрат"
```
